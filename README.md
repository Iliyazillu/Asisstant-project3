# 🤖 Auto-Reply Bot Collection

This repository contains two Python-based bots:

1. **WhatsApp Auto-Reply Insult Bot**
2. **Voice-Controlled Assistant**

---

## 📁 Project 1: WhatsApp Auto-Reply Insult Bot (`Project_4.py`)

A fun auto-responder for WhatsApp Web that replies to incoming messages with themed roast messages using text and speech.

### ✨ Features
- Connects to WhatsApp Web via Chrome
- Supports themes: `default`, `illu`, `roasty`, `romantic`
- Speaks replies using `pyttsx3`
- Fun, roast-style responses

### 🚀 Run It
```bash
pip install selenium pyttsx3 webdriver-manager
python Project_4.py
```

---

## 📁 Project 2: Voice Assistant (`project_3.py`)

A voice assistant that listens to your commands and opens specific websites like YouTube, Instagram, Google, etc.

### ✨ Features
- Voice recognition using `speech_recognition`
- TTS using `pyttsx3`
- Supports custom commands (e.g., open YouTube, play songs)

### 🚀 Run It
```bash
pip install speechrecognition pyttsx3
python project_3.py
```

> Make sure your microphone is working before you start!
